Class MsSqlDriver

The dibi driver for MS SQL database.

Driver options: - host => the MS SQL server host name. It can also include a port number (hostname:port) - username (or user) - password (or pass) - database => the database name to select - persistent (bool) => try to find a persistent link? - resource (resource) => existing connection resource - lazy, profiler, result, substitutes, ... => see Dibi\Connection options
